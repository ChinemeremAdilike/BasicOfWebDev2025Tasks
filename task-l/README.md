# RentIt Car Rentals – Task L

This folder contains the final refined version of the RentIt Car Rentals website, combining the landing page, catalog page, order page, and confirmation page into one coherent site.

## Pages
- `index.html` → Landing page
- `catalog.html` → Catalog page
- `order.html` → Order form
- `thanks.html` → Confirmation page after submitting the order form

## Change Log (Improvements from Tasks I–K)
- **Unified navigation bar** across all pages:
  - Links styled with uppercase text and hover color effects (no underlines).
  - Consistent header/footer design on every page.
- **Relative paths fixed** so all pages link correctly inside folder `l`.
- **Visual identity improved**:
  - Shared color palette (blue/white/gray).
  - Consistent typography, spacing, and button styles.
  - Catalog cards and order form styled with rounded corners and shadows.
- **Responsiveness enhanced**:
  - Catalog grid adapts to phone, tablet, and desktop.
  - Order form fields stack neatly on smaller screens.
- **Accessibility improvements**:
  - Semantic HTML structure (`header`, `nav`, `main`, `section`, `footer`).
  - Labels tied to inputs with `for` + `id`.
  - Alt text added for all images.
- **Form validation refined**:
  - Start date cannot be in the past.
  - End date must be after the start date.
  - Inline error messages added for clearer feedback.
- **Media integration**:
  - Catalog includes `example.jpg` and other vehicle images.
  - Promotional video with multiple formats (`mp4`, `webm`) and captions (`vtt`).
- **HTML validation**:
  - All pages tested with W3C Validator and adjusted for clean, valid markup.
